RoanSense

Potholes and poor road conditions pose significant risks to road safety, causing accidents, vehicle damage, and traffic delays. According to the CAA (Canadian Automobile Association), poor-quality roads cost Canadian drivers approximately $3 billion annually (CPCS, 2021). Traditional methods of detecting and repairing road damage are inefficient, leading to increased costs and road deterioration.

Current approaches rely heavily on periodic inspections and driver reports, which are neither timely nor comprehensive. There is a crucial need for advanced monitoring and detection systems utilizing AI and real-time data to efficiently identify and address road issues. This project aims to develop and implement such technologies to improve road maintenance and reduce associated costs.

For the POC (Proof of Concept) phase, we have developed three different services for our prototype project. The first is a Streamlit application where we apply our model algorithm. Users can upload images or videos, and the service will begin detecting poor road conditions. If any poor road conditions (such as potholes and cracks) are detected, it will automatically send an HTTP request to execute the API service we developed with Python Flask. This service includes a temporary database to store the data (detection), supporting CRUD (create, read, update, delete) operations. The API service uses token-based authentication to prevent unauthorized requests. Since this is the POC phase, implementing a more robust database, such as MySQL or SQL Server, is part of our future work.
The last service is a web application built with Vue.js, which serves as the data showcase platform. On this platform, we can observe the distribution and severity of poor road conditions in specific regions.
